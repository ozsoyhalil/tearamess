    GREEN phase — implement lists.ts with these functions:

    getUserLists(userId, includePrivate = false):
    - Queries lists table: .eq('user_id', userId)
    - Orders: .order('is_wishlist', { ascending: false }).order('created_at', { ascending: false })
    - If !includePrivate: .eq('is_public', true)
    - Select includes: 'id, name, description, is_public, is_wishlist, created_at, list_items(count)'
    - Returns { data: List[], error: null } or { data: null, error: message }

    getOrCreateWishlist(userId):
    - First: .select('*').eq('user_id', userId).eq('is_wishlist', true).single()
    - If found: return existing row
    - If PGRST116 (not found): insert { user_id, name: 'Gideceğim Yerler', is_public: true, is_wishlist: true }
    - If other error: return { data: null, error: message }

    createList(userId, name, description?):
    - Insert { user_id, name, description: description ?? null, is_public: true, is_wishlist: false }
    - .select().single()
    - Returns { data: List, error: null }

    deleteList(listId):
    - .delete().eq('id', listId)
    - Returns { error: null } or { error: message }

    addPlaceToList(listId, placeId):
    - Insert into list_items { list_id, place_id }
    - On unique constraint violation (duplicate): treat as success (no-op)
    - Returns { error: null }

    removePlaceFromList(listId, placeId):
    - .delete().eq('list_id', listId).eq('place_id', placeId)
    - Returns { error: null }

    addToWishlist(userId, placeId):
    - Calls getOrCreateWishlist(userId) to get/create the wishlist list id
    - Then calls addPlaceToList(wishlist.id, placeId)
    - Returns { error: null } or { error: message }

    removeFromWishlist(userId, placeId):
    - Calls getOrCreateWishlist(userId) to get wishlist id
    - Then calls removePlaceFromList(wishlist.id, placeId)
    - Returns { error: null }

    isPlaceInWishlist(userId, placeId):
    - getOrCreateWishlist(userId) → wishlist.id
    - .select('list_id', { count: 'exact', head: true }).eq('list_id', wishlistId).eq('place_id', placeId)
    - Returns { data: boolean (count > 0), error: null }

    updateListName(listId, name):
    - .update({ name }).eq('id', listId)
    - Returns { error: null }

    updateListPrivacy(listId, isPublic):
    - .update({ is_public: isPublic }).eq('id', listId)
    - Returns { error: null }

    getListById(listId):
    - .select('*').eq('id', listId).single()
    - If PGRST116: return { data: null, error: null } (not found / private — don't leak existence)
    - Returns { data: List | null, error: string | null }

    getListItems(listId):
    - Join query: .select('list_id, place_id, created_at, places(id, name, category, neighborhood)')
    - .eq('list_id', listId).order('created_at', { ascending: false })
    - Returns { data: ListWithPlaces[] | null, error: string | null }
    - ListWithPlaces = ListItem & { places: Pick<Place, 'id' | 'name' | 'category' | 'neighborhood'> | null }

    REFACTOR phase:

    - Ensure all returns follow { data, error } contract consistently
    - No function exceeds 30 lines
