  <files>src/components/ListItemSelector.tsx</files>
  <action>
Create `src/components/ListItemSelector.tsx`. A popover/dropdown anchored near its trigger button, showing the user's lists as checkboxes.

Props: `{ placeId: string, userId: string, isOpen: boolean, onClose: () => void }`

Behavior:
- `if (!isOpen) return null`
- On open (useEffect when isOpen → true): call `getUserLists(userId, true)` (includePrivate=true — user needs to see own private lists too)
- Also on open: for each list, check if place is already in it via a `list_items` query. Simplest approach: fetch `getListItems(listId)` is expensive — instead, store membership as a `Set<string>` (listIds where placeId already exists). Use a single query: `supabase.from('list_items').select('list_id').eq('place_id', placeId)` to get all lists containing this place for this user's lists. Filter to only the user's lists.

Rendering:
- Backdrop: fixed inset-0 z-40 (transparent), click → onClose
- Popover panel: absolute or fixed positioned, z-50, white bg, rounded-lg, shadow-xl, max-w-xs, max-h-64, overflow-y-auto
- Title row: "Listeye Ekle" text + X close button (onClose)
- List items: each row has checkbox + list name + place count (from item_count). Wishlist row shown first (is_wishlist=true stays at top per getUserLists ordering).
- Checkbox checked = placeId in list's items. Click: optimistic toggle (update Set), then call addPlaceToList or removePlaceFromList. Revert on error.
- Loading state: "Listeler yükleniyor..." text
- Empty state: "Henüz liste yok." text

Note: Use `supabase` directly in this component ONLY for the membership check query (the place-in-list check spans across all of user's lists — not a single list operation). All other calls go through the service layer.

Actually, to avoid direct Supabase usage in components (Phase 1 decision enforced strictly), add a `getPlaceListMembership(userId, placeId)` helper to lists.ts instead. This function returns a Set of list IDs (or an array of list_ids) where the given place exists, filtered to the user's lists.

Named export: `export function ListItemSelector`.
